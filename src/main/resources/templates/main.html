Board Games Online

What is this site?

This is a multiplayer board game site that I made over the summer. You can easily join games with anyone else on the site and play with them in real time. Currently Checkers, Tic Tac Toe, and Connect4 are supported.

How do I use this?

You can use the chatbox on this page to communicate. Use the navigation bar at the top of the page to navigate the site. Go to Play to find, join, or create games. Go to Reconnect to go to your latest game. More specific help is listed at the Help page.

How was this made?

The site is currently hosted on Heroku for free. The backend is a Spring Boot application. The front end is made without templates with Materialize. Sockets are used to communicate back and forth with the server.

More Info

I made this site with the intent of creating a very general purpose model for implementing board games. Rendering a more complicated game is tricky with the current system but any game with a grid system should be very easy to implement without changing any front-end code. The games are rendered from a .json file storing the necessary information. See the checkers file to see an example. I tried to include the means to display other parts of a game with predefined 'scoreboards.' The scoreboard data is sent alongside the board data. I tried to use these in the current games. TicTacToe has a timer scoreboard and checkers has a display of the current players color and counters for the amount of pieces taken. Overall, I'm satisfied with how this turned out as a completed project.
